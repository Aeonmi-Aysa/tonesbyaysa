USER DATA FLOW:

User Signs Up:
1. Email + password entered
2. Supabase creates auth.users entry
3. Trigger auto-creates profiles entry
4. Email sent for confirmation
5. User clicks link to verify
6. User can login now

User Starts Trial:
1. App calls Supabase
2. Records trial start date
3. subscription_tier = 'weekly'
4. User gets full access
5. 7 days later: auto-downgrade

User Purchases:
1. RevenueCat processes payment
2. App confirms entitlement
3. App updates Supabase
4. subscription_tier = 'weekly'/'lifetime'
5. Subscription persists
