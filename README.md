# Tones by Aysa

Multi-platform frequency wellness application.

## Mobile App (React Native)
- Location: /mobile
- Platforms: iOS, Android
- Tech Stack: React Native, Expo, TypeScript
- Features: 420+ frequencies, 86 baths, manifestation, composer

### Quick Start (Mobile):
```bash
cd mobile
npm install
expo start
```

## Browser App (Progressive Web App)
- Location: /browser
- URL: https://tonesbyaysa.netlify.app
- Tech Stack: HTML, JavaScript, Web Audio API
- Features: Manifestation hub, frequency composer, favorites

### Quick Start (Browser):
```bash
cd browser
# Automatically deploys via Netlify on push to master
```
